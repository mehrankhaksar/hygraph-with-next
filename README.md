# 🎮 GameZone

A modern gaming blog built with **Next.js 16**, **Hygraph CMS**, **GraphQL Request**, **React 19**, **Tailwind CSS 4**, and **TypeScript**.

## 🚀 Features

- Next.js 16 App Router
- Hygraph CMS integration
- GraphQL Request for GraphQL queries & pagination
- Server Actions for secure data fetching
- Cursor-based pagination (Load More)
- Tailwind CSS 4 for styling
- Fully type-safe with TypeScript
